# Nile v0.16.10

## Highlights

- Add an optional macOS menu bar usage ticker so Nile can show quota summaries for selected agents directly in the top bar.
- Let you choose between the compact Nile menu bar entry and the new ticker mode from Settings.
- Make ticker selection stick across app restarts and mode switches, with a sensible default when quota data is available.

## Included Changes

- Each supported agent quota row in the menu bar menu can now be included in or removed from the ticker.
- Ticker mode falls back to the regular Nile menu bar entry when the selected agents do not currently expose usable quota data.
- The default inferred ticker selection now behaves correctly on first click instead of requiring an extra toggle.
- Added localized menu bar settings copy for the shipped desktop languages and removed the unused ticker hint text from the settings catalog.
