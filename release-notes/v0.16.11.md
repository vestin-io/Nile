# Nile v0.16.11

## Highlights

- Added a menubar usage ticker with per-agent inclusion controls and a compact picker-oriented menu flow.
- Added Gemini CLI quota reading and Gemini model detection in saved connections.
- Improved the README previews with a larger animated desktop demo and expanded menubar visuals.

## Included Changes

- You can now switch the menu bar between a compact app entry and a quota ticker, then choose exactly which agents appear in the ticker.
- Gemini connections now surface usage windows and detected models through Nile's shared connection flows.
- Applying a saved Gemini model now updates Gemini CLI's default model for newly launched sessions.
- The connections list now labels Gemini connections correctly instead of falling back to an unknown provider label.
- README previews now include an animated desktop demo and a clearer two-image menubar section.

## Upgrade Notes

- Gemini usage and model detection still depend on the local Gemini CLI session being fresh. If the saved Gemini OAuth token has expired, refresh that local session and reopen Nile.
