# Nile Desktop v0.15.4

## Highlights

- Fixed the signed macOS release build by pinning the Electron runtime version used by electron-builder.
- Carries forward the v0.15.3 release lockfile fix and the macOS 12 compatible universal keychain helper.

## Included Changes

- Set the desktop Electron dependency to a fixed `36.9.5` version instead of a floating range.
- Added an explicit `electronVersion` to the desktop electron-builder config so CI does not have to infer the runtime from workspace-installed modules.
- Keeps the release workflow compatible with clean `npm ci` installs on GitHub Actions.

## Upgrade Notes

- Intel and Monterey users should install v0.15.4 or newer before re-testing saved OpenAI/Codex session connections.
