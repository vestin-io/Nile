# Nile Desktop v0.15.3

## Highlights

- Fixed the desktop release dependency lock so the signed macOS release workflow can install dependencies with `npm ci`.
- Carries forward the v0.15.2 keychain helper compatibility fix for Intel Macs and macOS 12 Monterey.

## Included Changes

- Synced the root lockfile with the current workspace dependency graph so GitHub Actions no longer fails on missing `esbuild` packages during release setup.
- Keeps the bundled keychain helper as a universal macOS binary with a macOS 12.0 deployment target.
- Keeps improved credential-sync diagnostics for helper startup failures.

## Upgrade Notes

- Intel and Monterey users should install v0.15.3 or newer before re-testing saved OpenAI/Codex session connections.
