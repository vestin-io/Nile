# Nile Desktop v0.17.8

## Highlights

- Fixed Codex desktop profile apply so a setup home override now reapplies the selected saved connection, instead of leaving the new home on stale auth state.
- Fixed the Agents refresh flow so usage now refreshes immediately from the source of truth instead of reusing a fresh cached value.
- Expired desktop sessions now stay silent during background and bulk refreshes. Nile retries same-identity session sync automatically, then shows the unauthorized state in the UI instead of unexpectedly opening login windows.

## Included Changes

- Improved desktop usage recovery for saved OpenAI and Gemini session connections when the current local setup already matches the same account.
- Tightened the refresh pipeline so explicit usage refreshes can bypass cache without also triggering interactive reauthentication flows for unrelated connections.
- Added regression coverage for same-connection profile apply, forced settings refresh behavior, and silent unauthorized recovery.

## Upgrade Notes

- If a saved session is still expired after Nile retries the current local session silently, use the UI reauth path for that specific connection or import the refreshed local session again.
