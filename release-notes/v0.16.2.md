# Nile Desktop v0.16.2

## Highlights

- Polished the in-app update prompt so it feels lighter and easier to scan.
- Fixed a desktop shutdown crash that could appear while update state was still changing in the background.

## Included Changes

- Simplified the update notification card with tighter layout, shorter copy, and clearer download/install states.
- Hardened the desktop shell so updater events no longer try to talk to windows that have already been closed.
